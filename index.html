An Intrusion Detection tool for Unix Clones

GitHub

Download

This is the website for rkhunter.sh a tool a command line utility to check computers running UNIX (or clones) for known root kits, other unwanted tools and changed files.

The project was started in 2003, by Michael Boelen, and from 2006 was continued by the Rootkit Hunter Development Team led by John Horne and unSpawn.

Since December 2023 it has been sponsored by Dogsbody Technology.
