The Rootkit Hunter Project

Welcome to the Rootkit Hunter Project Site

About

This is the website for rkhunter.sh a tool a command line utility to check computers running UNIX (or clones) for known root kits, other unwanted tools and changed files. The project was started in 2003, by Michael Boelen, and from 2006 was continued by the Rootkit Hunter Development Team led by John Horne and unSpawn. Since December 2023 it has been sponsored by Dogsbody Technology

Rootkit Hunter current release

Rootkit Hunter is currently at version 1.4.6, released on February 20th 2018. We are working on addressing issues brought over from SourceForge, and new releases will be found on on the project’s GitHub Release Page

Downloading and running Rootkit Hunter

To run Rootkit Hunter please install or upgrade to Rootkit Hunter version 1.4.6, and read the README file.

  • If you have any questions about what Rootkit Hunter reports, or if you encounter a runtime or configuration problem, then please first consult the Rootkit Hunter installation tutorial (if applicable), the Rootkit Hunter FAQ and the rkhunter-users mailing list archives.
  • If your question is not answered in the FAQ or the mailing list archives, please join (subscribe) to the rkhunter-users mailing list and ask.
  • If you would like to be kept informed of updates we also have a (very) low volume rkhunter-announce mailing list.
  • If you would like to report a bug, downstream (package maintainer) changes or supply a patch, then please use the Rootkit Hunter bug tracker. If unsure, then please first check on the rkhunter-users mailing list.
